UDOSURVEY

Resumen

¿Cómo puede UDOSURVEY revolucionar la forma en que las personas crean encuestas, ofreciendo una experiencia rápida, efectiva y segura?

UDOSURVEY es una aplicación innovadora diseñada para simplificar y mejorar el proceso de creación de encuestas. Con un enfoque en la rapidez, efectividad y seguridad, UDOSURVEY proporciona a los usuarios una herramienta ágil y confiable para recopilar y analizar datos de encuestas. Algunas de las características clave de esta aplicación son:

  • Una interfaz de usuario intuitiva que permite a los usuarios crear y personalizar encuestas en poco tiempo.
  • Una amplia gama de tipos de preguntas y plantillas de encuesta para adaptarse a diversas necesidades de investigación.
  • Medidas de seguridad sólidas que protegen la información personal de los participantes en las encuestas y garantizan la confidencialidad de los datos recopilados.
  • Herramientas avanzadas de análisis de datos que permiten a los usuarios extraer conclusiones valiosas de las respuestas de las encuestas y tomar decisiones informadas.

    • Objetivos generales

    • Desarrollar una aplicación que permita tabular información veraz, ágil y confiable, detectando incongruencias, falsedad y duplicidad de la información recopilada.

    Tecnología utilizada

    Las tecnologias utilizadas hasta el momento son:

    • MySQL: La creacion, edición, borrado y consulta de datos es fundamental es este proyecto, por lo cual una herramienta de base de datos es necesaria para el funcionamiento de la app como lo es MySQl, gracias a esto se ha podido dar la creación del primer diagrama E/R.
    • Github: Una plataforma la cual albergue los documentos del desarrollo de la app es parte fundamental siendo Github el metodo utilizado para desplegar el contenido realizado respecto a la app. En este repositorio podemos visualizar el diagrama E/R, el documento escrito de la hipotesis a desarrollar y una pagina preliminar en HTML del proyecto.

    Se ha dado manejo de otras herramientas tecnológicas en este caso para la recolección de datos por medio de una encuesta de prueba en la cuál contiene una serie de 6 preguntas para saber la opinión de los usuarios sobre las encuestas.
